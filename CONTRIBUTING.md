# Contributing

Issues, feature requests, and pull requests are welcome.

## Reporting Issues

Open an issue at [github.com/roman-rr/trading-skills/issues](https://github.com/roman-rr/trading-skills/issues) with:
- What you expected vs what happened
- Your install method (npx, plugin, manual)
- Any error messages

## Skill Structure

```
trading-signals/
├── SKILL.md              # Main skill instructions
├── README.md             # Short description
├── agents/openai.yaml    # OpenAI Codex config
├── assets/icon-small.svg # Marketplace icon
└── references/           # Supplementary docs
    └── SIGNAL-FORMAT.md
```

## Pull Requests

1. Fork the repo
2. Create a branch (`git checkout -b fix/description`)
3. Make changes
4. Open a PR against `main`
